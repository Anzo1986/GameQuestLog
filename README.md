## ğŸš€ Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Anzo1986/GameQuestLog.git
    cd GameQuestLog
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```

4.  **Build**
    ```bash
    npm run build
    ```
